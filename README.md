# B-mode-Imaging
Code for B-mode imaging base on Ultrasound image

### File Tree
```
├── data
│   ├── personal
│   ├── phantom
│   └── parameters.json
└── b-mode-imaging.ipynb
```

### Data
- Ultrasound image data is in `data` folder
    - `personal`: my neck data
    - `phantom`: reference data
    - `parameters.json`: metadata of image
